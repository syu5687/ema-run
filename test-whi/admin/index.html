編集を反映する手順（Phase 1）

  1. 各タブで内容を編集する
  2. 「JSONを書き出す」でファイルをダウンロードする
  3. ダウンロードしたファイルで src/data/ の同名ファイルを置き換える
  4. いつもの手順（build → dist 反映 → push）で公開する

注意

  • この画面はまだサーバーに保存しません。編集したら都度ダウンロードしてください。
  • ページを再読み込みすると、ダウンロードしていない編集内容は消えます（タブの切り替えでは消えません）。
  • コンタクト履歴は、保存処理とログイン（認証）を実装してから有効化します。
